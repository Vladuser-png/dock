# dock

Лендінг для мобільного застосунку онлайн-запису до лікаря.

**Ринок:** Україна · **Платформа:** Desktop → Mobile adaptive · **Стек:** Next.js 14 + Vercel

---

## Структура репо

| Папка | Що всередині |
|-------|-------------|
| [`research/`](./research/) | Конкурентний аналіз, інсайти, референси |
| [`research/screens/`](./research/screens/) | Скріншоти конкурентів і референсів |
| [`wireframes/`](./wireframes/) | Вайрфрейми лендінгу (lo-fi → mid-fi) |
| [`concept/`](./concept/) | Концепт і напрямки візуального стилю |
| [`tokens/`](./tokens/) | Дизайн-токени: кольори, типографіка, відступи |
| [`components/`](./components/) | Атомарні компоненти (кнопки, картки, форми) |
| [`design-system/`](./design-system/) | Зібрана дизайн-система |
| [`handoff/`](./handoff/) | Фінальні файли для передачі в розробку |

## Продукт

Детальний бриф → [`CLAUDE.md`](./CLAUDE.md)

**Ключові фічі застосунку:**
- Онлайн-запис із вибором слоту
- Електронна черга
- Картки лікарів
- Медична карта пацієнта
- Відеоконсультації

**Аудиторія лендінгу:** пацієнти + клініки + лікарі
